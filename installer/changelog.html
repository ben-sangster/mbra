Version 2.3.0 (2010-10-20)

New Features

  • Added Flow Consequence to analysis for simulation and calculation tabs.
  • Added Consequence Reduction, Threat Maximization, and enhanced Risk Minimization to the NA calculations.
  • Added cascade failure simulation to the tools and graphs palettes.
  • Monetary Units have been added to the MBRA Preferences menu. These add a text label next to all monetary units for user reference. They are solely text-based, and do not affect any calculations.

Changes

  • Modified the graph so that at its default bar width, the bars will not label bars that have 100% height. This was done to prevent text label overlap due to the decreased width of the bars.
  • Re-ordered the default order of NA data table columns.
  • Calculations performed in the "Calculations" tab of the NA tools window will now only update when the Calculate button in that window is on. This includes objective graph updates and rankings on nodes in the main window.
  • Contagiousness is now a "Weight By" option, rather than an "Objective Function".

Bugs fixed

  • Fixed FT paths in data table.
